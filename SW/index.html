Stars Warcraft

Utilize a tecla W para movimentar a nave para cima, a tecla S para movimentar a nave para baixo e a tecla D para atirar. A cada 3 inimigos perdidos, são tomados 500 pontos, e caso os pontos cheguem a 0, uma vida será tirada a cada 3 inimigos perdidos! Boa Sorte!!!
